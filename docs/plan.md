## 決めること一覧

### 1. 開催時期

2026年 秋〜冬（10月・11月・12月）のいずれかの土曜日。

- **木更津・千葉**: 高専の地元。「あの場所に戻る」感がある。アクセスはやや限定的
- **東京**: アクセス良好で集まりやすい。二次会の選択肢も多い

### 3. 声をかけるバンド・メンバーのリストアップ

- 対象学年: 何学年分を対象にするか
- バンド数の目標: 5〜10バンド
- 出演者: 20〜40人（バンドあたり3〜5人）
- 観覧者: 10〜20人（バンドメンバーの友人・パートナーなど）
- 合計: 50〜60人
- 各バンドへの連絡方法・声かけの順番

---

### 4. 予算・会費

#### 主な費用項目

- 会場レンタル費
- PA・照明（会場備え付けの場合は不要）
- 追加機材レンタル（必要に応じて）
- 二次会費用
- 雑費（記念品、撮影関連など）

#### 会費の考え方

- 出演者と観覧者で傾斜をつけるか
- 一律にするか
- 一人あたりの目安を算出

## スケジュールマイルストーン

幹事チーム結成後、本番までのロードマップ。

### Phase 1: 企画確定（2〜3月）

- [ ] 幹事チーム結成（3〜5人）
- [ ] 開催時期の決定（月 + 候補日の絞り込み）
- [ ] 会場の方向性を決定（エリア: 千葉圏 or 東京）
- [ ] 予算感の合意（一人あたりの目安）

### Phase 2: 基盤固め（4〜5月）

- [ ] 会場の下見・予約確定
- [ ] 出演バンドの声かけ開始
- [ ] 参加意向の回収（第一次締め切り）
- [ ] 連絡用グループ作成（LINE等）

### Phase 3: 出演者確定（6〜7月）

- [ ] 出演バンド確定（セットリスト提出依頼）
- [ ] タイムテーブル草案作成
- [ ] 会費確定・徴収方法の決定
- [ ] 観覧者への告知開始

### Phase 4: リハーサル期間（8〜9月）

- [ ] 各バンド自主リハーサル
- [ ] セットリスト最終提出
- [ ] タイムテーブル確定
- [ ] 当日の運営マニュアル作成
- [ ] 二次会会場の予約

### Phase 5: 直前準備（本番1ヶ月前〜）

- [ ] 最終参加者確認
- [ ] 会費徴収
- [ ] 機材・備品の最終確認
- [ ] 当日の役割分担確認
- [ ] リマインド連絡

### Phase 6: 本番当日

- 10:00〜12:00 機材搬入・セッティング
- 12:00〜16:00 リハーサル（各バンド30分）
- 16:00〜16:30 休憩・最終調整
- 16:30〜17:00 開場・受付
- 17:00〜17:10 開会挨拶
- 17:10〜20:10 ライブ本番（8バンド x 約20分 + 転換5分）
- 20:10〜20:30 全員セッション
- 20:30〜20:45 記念撮影・クロージング
- 21:00〜23:00 二次会
