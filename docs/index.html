Concept

卒業から20年。

高専時代のバンドを、もう一度やる。
ライブハウスを一日借り切って、本番をやる。
打ち上げまで含めて、まるごと一日。

20年の節目に、もう一度火を灯してみませんか。

Schedule

2026年 秋〜冬で検討中

Oct.
気候が良い 会場取りやすい 準備期間短め

秋の好天で過ごしやすい。オフシーズンで会場も安い。ただし準備期間は約8ヶ月。

Nov.
予定を合わせやすい バランス良し

気候・コスト・準備期間のバランスが良い。3連休（勤労感謝の日）前後が狙い目。

Dec.
準備期間は十分 忘年会の盛り上がり 会場競争激化

忘年会シーズンで「同窓会」感は出しやすい。ただし会場予約は早め必須、宿泊コストも高め。

Next Actions

これから決めること

01

参加候補メンバーの洗い出し

何学年分を対象にするか、何バンドを目標にするか、観覧者の範囲をどうするか。まずは声をかける候補をリストアップし、参加意向・日程希望と合わせて希望する役割（企画統括 / 会場・日程調整 / 出演者連絡 / 当日運営 / 会計など）もアンケートで確認する。

02

メンバーの意向を聞いて日程を確定する

候補メンバーに参加意向と日程の希望を確認し、それを踏まえて開催日を決定する。
